## Description

Shoplet is a fictional ecommerce website built with react and tailwind.

## Installation

1. Clone the repository
```bash
git clone https://github.com/Feycalion/shoplet
```
2. Install dependencies
```bash
npm install
```
3. Run application
```bash
npm run dev
```
